# Frontend README

## Setup Instructions

-   I created the frontend using React with Vite.
-   Install dependencies using: `pnpm install`.
-   Run the frontend locally using: `pnpm dev`.

## Environment Variables

-   Create a `.env` file in the root.
-   Example variables:
    -   `VITE_API_BASE_URL=http://localhost:5000`
    -   `VITE_NODE_ENV=development`

## Features

-   User registration and login.
-   Loan application form with officer selection and interest rate
    input.
-   Dashboard to track loan status (pending, approved, rejected).
-   Interactive and responsive UI.

## How to Run

-   Clone the repository.
-   Install dependencies.
-   Set up `.env` with API URL.
-   Start the frontend: `pnpm dev`.
-   Open `http://localhost:5173` in your browser.
